# Prerequisites for iOS App

These tasks must be completed by the user before or during implementation.

## Before Starting

- [x] Create Xcode project with iOS 26 deployment target, SwiftUI app lifecycle, and bundle identifier — created at `Flux/`
- [x] Add App Group capability in Xcode — configured as `group.me.nore.ig.flux`
- [x] SwiftData is already imported in the project template
- [x] Ensure the Flux backend (Lambda API) is deployed and accessible for integration testing

## During Implementation

- [x] Add the App Group identifier to the Keychain access group in entitlements — `group.me.nore.ig.flux` configured
